## 2025-04-02
- Rubyの**宇宙船演算子（＜＝＞）**は、左辺と右辺の大小関係に応じて異なる値を返す比較演算子。三方比較演算子とも呼ばれる。
【宇宙船演算子の使い方】
左辺と右辺を比較して、左辺が大きければ正の整数、小さければ負の整数を返す。
双方等しければ 0、比較できなければ nil を返す。

## 2025-08-09
<<演算子の基本
用途: Rubyでは、<<は「シフト演算子」としても使われるが、主に配列や文字列に要素を追加するために使われる。

配列への追加
使い方: 配列に要素を追加するのに使われる。例えば、以下のように使う。
```
array = []
array << 1   # arrayは [1] になる
array << 2   # arrayは [1, 2] になる
```

文字列への追加
使い方: 文字列に別の文字列を追加するのにも使われる。
```
string = "Hello"
string << " World"  # stringは "Hello World" になる
```

特徴
破壊的操作: <<は対象のオブジェクトを直接変更するので、元の配列や文字列が変わることに注意が必要。
